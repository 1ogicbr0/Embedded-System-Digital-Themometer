# Embedded-System-Digital-Themometer
Digital Themometer for measuring Thermistor + using push buttons.\n
> An Electrical Engineering Project.
We were supposed to read thermistor display reading on thermometer. <
# Components:
  --> MCU (pic16f882)
  --> LCD (LM016L)
  --> Thermistor (NTC)
  --> Resistors
  --> 5V battery
  --> Push buttons
